# Circle punch
It's simple browser multiplayer game powered by [Rust](https://www.rust-lang.org/) language.

#### How to run with local server
1. Install `Rust` (see instruction on the site)
2. Start server
```
cargo run
```
3. Open `client_src/index.html`
